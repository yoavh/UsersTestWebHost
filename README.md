### Requirements

* Node 12
* Git

### Installation

clone repo

`git clone https://github.com/yoavh/UsersTestWebHost.git`

install npm 

`npm install`

### Start Server with npm

  - `npm start`
  - with inspect 'npm run inspect'
  - with nodemon and inspect 'npm run watch'
